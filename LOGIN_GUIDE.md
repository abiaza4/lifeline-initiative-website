# Admin Dashboard Login Guide

## How to Access the Admin Dashboard

### Login URL
Navigate to: `/admin/login` or click the Admin Login link in your browser

### Demo Credentials

**Email:** `admin@liss-southsudan.org`  
**Password:** `LISS@2024`

### Login Steps

1. Go to the login page: `https://your-domain.com/admin/login`
2. Enter the email: `admin@liss-southsudan.org`
3. Enter the password: `LISS@2024`
4. Click "Sign In"
5. You will be redirected to the dashboard

### Dashboard Navigation

Once logged in, you can access all admin features:

- **Dashboard** - View statistics and overview
- **Programs** - Manage program content
- **News** - Create and publish articles
- **Form Submissions** - Review inquiries and applications
- **Donations** - Track donations received
- **Team Members** - Manage leadership information
- **Transparency** - Upload financial reports

### Session Management

- Sessions expire after **24 hours** of inactivity
- You will be automatically logged out and returned to the login page
- Click "Sign Out" in the admin sidebar to manually logout

### Features

- Full CRUD operations (Create, Read, Update, Delete)
- Search and filter capabilities
- Export and import data
- Responsive mobile interface
- Dark/Light theme support
